SYSTEMS [[15 THE NAME MACHINE]]
DOMAIN [[THE LANGUAGE]]

DEFINITION:  
  
A defined grouping, class, division, or container used to organize entities, concepts, objects, people, events, or phenomena according to shared characteristics, attributes, relationships, or criteria.  
  
FUNCTION:  
  
Provides the basic units through which classification systems organize reality into recognizable and communicable forms.  
  
EXAMPLES:  
  
- Species categories  
- Product categories  
- Personality categories  
- Political categories  
- Occupational categories  
- Educational categories  
- Cultural categories

NOTES:  
  
Categories allow complex realities to be divided into manageable groups. The characteristics used to define a category may vary depending on the purpose, framework, culture, institution, or classification system being employed.  
  
CONTROL TEXT OBSERVATION:  
  
Before something can be named, it is often placed into a category. Categories function as conceptual containers that organize perception by grouping diverse entities under shared classifications, allowing reality to become structured, navigable, and communicable.
