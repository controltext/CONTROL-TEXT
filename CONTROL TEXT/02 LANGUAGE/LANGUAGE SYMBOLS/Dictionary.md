SYSTEM [[17 LANGUAGE ENGINE]]
DOMAIN [[THE LANGUAGE]]

DEFINITION:  
  
A structured reference system that records, organizes, defines, and preserves words, terms, symbols, and their associated meanings within a language.  
  
FUNCTION:  
  
Provides a framework through which words can be standardized, interpreted, communicated, referenced, and transmitted across individuals, groups, and generations.  
  
EXAMPLES:  
  
- Language dictionaries  
- Technical dictionaries  
- Specialized glossaries  
- Lexicons  
- Vocabulary references  
- Terminology databases  
- Symbol dictionaries

NOTES:  
  
Dictionaries function as repositories of linguistic meaning by associating words with definitions, usage, pronunciation, context, and interpretation. Different dictionaries may reflect different linguistic traditions, disciplines, cultures, or periods of time.  
  
CONTROL TEXT OBSERVATION:  
  
Language requires shared meaning to function effectively. A dictionary serves as a linguistic reference system that stabilizes communication by connecting symbols to definitions, allowing words to be interpreted consistently across individuals, institutions, and generations.
